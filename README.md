# Gym Attendance & Workout Activity Prediction 🏋️‍♂️

## 📌 1. ปัญหาและที่มา (Problem Statement)
โปรเจคนี้จัดทำขึ้นเพื่อวิเคราะห์ข้อมูลการเข้าใช้บริการยิมและพฤติกรรมการออกกำลังกาย 
เพื่อทำนาย [ โอกาสในการมาเข้ายิม ] 
ซึ่งจะช่วยให้เจ้าของธุรกิจยิมวางแผนทรัพยากรได้ดีขึ้น และสมาชิกสามารถติดตามเป้าหมายได้แม่นยำ

## 📊 2. เกี่ยวกับข้อมูล (Dataset)
ข้อมูลจาก Kaggle: [Daily Gym Attendance and Workout Activity Data]
